Crie uma classe chamada "FuncionarioComissionado" com ES6, que herdará os métodos da classe "Funcionario". A classe "Funcionario" deve ter os seguintes métodos: constructor(nome, numCTPS, salario) e mostrarDados() para retornar todos os dados de um funcionário. A classe "FuncionarioComissionado" deve ter os seguintes métodos: constructor(nome, numCTPS, salario, venda, comissao) e mostrarDados() para retornar todos os dados de um funcionário comissionado. Em seguida, crie um objeto de cada uma das classes e mostre os respectivos dados no browser.
